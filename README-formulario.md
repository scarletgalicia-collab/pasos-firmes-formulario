# Pasos Firmes — Formulario de Inscripción

Formulario de inscripción para la comunidad **Pasos Firmes** (Charcot-Marie-Tooth).

## 🌐 URL del formulario

Una vez activado GitHub Pages:
```
https://scarletgalicia-collab.github.io/pasos-firmes-formulario/
```

## 📁 Estructura

```
pasos-firmes-formulario/
├── index.html                   ← Formulario (renombrar unirme-pasos-firmes.html)
└── README.md
```

## 🚀 Publicación

1. Sube el archivo `unirme-pasos-firmes.html` a este repositorio
2. Renómbralo a `index.html`
3. Ve a Settings → Pages
4. Source: `main` branch / `(root)`
5. Save y espera 2-3 minutos

## 🔗 Repositorio relacionado

El sitio web principal está en:
- **URL:** `https://scarletgalicia-collab.github.io/pasos-firmes-web/`

Todos los botones "Volver al inicio" apuntan a esa URL.

## 📋 Contenido del formulario

4 pasos:
1. **Datos personales** (nombre, ubicación, email, parentesco)
2. **Perfil clínico** (tipo CMT, tiempo, síntomas, malformaciones)
3. **Tratamientos** (terapias, neurológicos, apoyos técnicos)
4. **Participación comunitaria** (círculo de autoayuda)

Incluye barra de progreso, validación HTML5 y pantalla de éxito.

## 📝 Notas técnicas

- **Codificación:** UTF-8
- **Responsive:** Sí
- **Logo:** Cargado desde `pasos-firmes-assets`
- **Sin backend:** Los datos se validan en frontend (no se envían a ningún servidor todavía)
